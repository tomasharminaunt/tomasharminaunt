Project Title: MealPal
Team Members: Eddie Lim, Joowon Chung, Tomas Harmina
------------------------------------------------------------


Introduction:
-----------------
MealPal is a web-based recipe recommendation platform that helps college students living off-campus find simple and
affordable meals based on the ingredients they already have, with college students as the primary users.
The application allows users to input ingredients they already have and receive simple, affordable recipe suggestions.
MealPal will be web-based, responsive, and accessible without installation.
________________________________________________________________________________________________________________________________________________________________________________________________________________________________


Mealpal Requirements
---------------------
Ingredient Input - Users can enter ingredients they have.
Recipe Suggestion - Suggests recipes based on input ingredients.
Recipe Details - Displays ingredients, steps, and cooking time.
Admin Management - Admin can manage recipe data.
Mobile Responsiveness - Supports multiple device screen sizes.
Performance - Search results load within 1 second.
Security - Admin functions protected with authentication.
Usability - Easy to use for non-technical users.
Backup - Daily backups of recipe data.
Session Timeout - Logout after 10 minutes of inactivity.
